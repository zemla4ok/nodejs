  • Macro tasks - executes only one at the iteration of loop

Node.js - Event Loop

  • Macro tasks (tasks queue): setTimeout, setInterval, setImmediate, requestAnimationFrame, I/O, UI rendering
  • Micro tasks - process.nextTick, Object.observe, Promises (resolved)
